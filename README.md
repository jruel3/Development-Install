#Code Development Install Automation Script for MacOS

This script installs the following packages:
```
 - XCode Command Line Tools
 - HomeBrew
 - Github CLI
 - JDK
 - Zoom
 - Node.js
 - pyenv
 - Python 3.9.10
 - Blender
 - Unity
 - Android Studio
 - Intellij-CE
 - VNC-Viewer
 - Libre Office
 - Visual Studio Code
 - Arduino IDE
```
