# Code Development Install Automation Script for MacOS

## Install 
`git clone --depth=1 https/github.com/jruel3/development-install`

`cd development-install`

`zsh ./install.sh`

## About

This script installs the following packages:

 - HomeBrew
 - Github CLI
 - JDK
 - Zoom
 - Node.js
 - pyenv
 - Python 3.9.10
 - Blender
 - Unity
 - Android Studio
 - Intellij-CE
 - VNC-Viewer
 - Libre Office
 - Visual Studio Code
 - Arduino IDE
